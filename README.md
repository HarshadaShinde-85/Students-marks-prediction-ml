## Students Marks Prediction - ML Project
This Machine Learning project predicts student marks based on the number of study hours using **Linear Regression**.

## Project Overview
The objective of this project is to build a regression model that can predict student marks based on how many hours they studied.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset Information
The dataset contains:
- Study Hours (Independent Variable)
- Marks (Dependent Variable)

## Project Workflow
1. Data Loading
2. Data Visualization
3. Data Splitting (Train-Test)
4. Model Training using Linear Regression
5. Model Prediction
6. Model Evaluation
